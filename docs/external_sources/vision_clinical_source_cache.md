# Vision Clinical Source Cache

This index documents repaired and migrated primary sources for vision conditions
that touch more than one accessibility axis, especially color, acuity, contrast,
visual fields, photophobia, and eye movement stability.

## Scope

These artifacts back the repo's blue cone monochromacy, contrast sensitivity,
low vision, nystagmus, and visual-field-loss research lanes.
